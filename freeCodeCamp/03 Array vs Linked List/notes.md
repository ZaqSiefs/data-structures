|                                  |      **Array**      | **Linked List**   |
|----------------------------------|:-------------------:|-------------------|
| **Cost of accessing an element** | O(1)                | O(n)              |
| **Memory requirements**          | Best for small data | Best for big data |
| **Cost of Inserting an element** | O(n)                | O(n)              |
| **Ease of use**                  | Better              | Worse             |
